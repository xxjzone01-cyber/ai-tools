## 🎯 域名配置步骤

### 步骤 1: GitHub Secrets配置

2. Settings → Secrets and variables → Actions
3. 添加两个Secrets

### 步骤 2: Cloudflare DNS配置
**耗时**: 3分钟

**耗时**: 2分钟
